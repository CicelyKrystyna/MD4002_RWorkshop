Installing R

Installing R#

  • Navigate to The Comprehensive R Archive Network (CRAN)

  • Click on the appropriate link to navigate to downloads R for your operating system.

    • If downloading on Windows you should select install R for the first time and then select the most recent version e.g. Download R-4.5.0 for Windows

    • If downloading on Mac you should choose the correct download for your system e.g. for Apple Silicon R-4.5.0-arm64.pkg.

    • There is also a Linux download if you prefer.

  • Navigate to your Downloads folder and initiate the download process following the installation instructions accepting recommended settings and default save locations.
